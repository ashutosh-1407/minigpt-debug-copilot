# Dataset Design

## Domain
Engineering debugging and backend troubleshooting

## Example Categories
- Python tracebacks
- FastAPI errors
- Docker/container issues
- SQL query mistakes
- API timeouts
- import/module path errors
- logging interpretation
- config mistakes

## Example Format
JSONL with:
- prompt
- response

## Goals
- narrow and high-signal
- useful for debugging-style answers
- enough variation for common backend issues

## Notes
Start small and clean before scaling dataset size.
